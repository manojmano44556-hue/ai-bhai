# AI Bhai: The Ultimate Free Multimodal AI Chef 👨🏽‍🍳

AI Bhai is a high-performance, **Zero-Key** personal cooking assistant. It combines the power of Multimodal Vision, Large Language Models (LLMs), and real-time AI image generation into a premium, cost-free package.

## 🚀 The "Zero-Key" Innovation
This project demonstrates how to build a professional-grade AI application without paying for expensive APIs:
- **Vision**: Uses `Moondream2` via Gradio for zero-cost ingredient recognition.
- **Intelligence**: Uses `Pollinations.ai` (Llama-3 powered) for storytelling and recipe generation.
- **Visuals**: Real-time step-by-step image generation using `Pollinations.ai`.
- **Languages**: Native support for **Hindi, Telugu, and English** (using transliterated script).

## ✨ Features
- **Multimodal Ingredient Scan**: Take a photo of your kitchen, and AI Bhai identifies every vegetable and spice.
- **Storytelling Cooking**: Each recipe is shared as a warm, engaging narrative.
- **Visual Step-by-Step**: Every single cooking instruction includes an AI-generated image.
- **Nutrition Analytics**: Instant calculation of calories, protein, carbs, and fats.
- **Premium Glassmorphism UI**: A stunning "Modern Dark Chef" interface with gold accents and high-end typography.

## 🛠️ Technology Stack
- **Frontend**: Streamlit + Custom CSS (Playfair Display & DM Sans)
- **AI Backend**: Gradio Client (Vision) & Pollinations API (Text)
- **Image Processing**: Pillow (PIL)
- **Cloud Hosting**: Optimized for Streamlit Community Cloud

## 📦 Quick Start
1. **Clone & Install**:
   ```bash
   git clone https://github.com/yourusername/ai-bhai-chef.git
   pip install -r requirements.txt
   ```
2. **Run (No API Key Required!)**:
   ```bash
   streamlit run app.py
   ```
